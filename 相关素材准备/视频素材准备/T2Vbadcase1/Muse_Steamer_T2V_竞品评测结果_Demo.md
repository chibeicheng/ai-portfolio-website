# 三、模型分析

## 1. Muse Steamer

### 画面质量（4.5）

**优点**

-   景深自然
-   金属材质表现良好
-   光影氛围较好

**问题**

-   局部边缘细节略软
-   暗部细节不足
-   构图表现一般

### 运动质量（4.2）

**优点**

-   手部运动连续
-   运镜稳定

**Bad Case**

-   手腕运动略机械
-   笔尖接触纸面缺少真实阻尼感
-   动作节奏略均匀

**归因**

-   运动合理性 → 人体运动异常
-   动作幅度符合度 → 动作幅度不足

### 指令遵循（4.7）

-   主体 ✔
-   动作 ✔
-   场景 ✔
-   镜头 ✔
-   细节：书写轨迹略简单

**综合评价**

整体能力均衡，但动作真实感仍有提升空间。

------------------------------------------------------------------------

## 2. Kling

### 画面质量（4.7）

**优点**

-   清晰度优秀
-   材质纹理丰富
-   景深控制优秀

**问题**

-   高光区域轻微过曝

### 运动质量（4.7）

**优点**

-   手部骨骼自然
-   写字动作稳定
-   运镜自然

**Bad Case**

-   镜头运动质量 → 轻微镜头漂移

### 指令遵循（4.6）

整体符合 Prompt，签名字体存在一定随机性。

**综合评价**

高画质、高运动质量，综合表现优秀。

------------------------------------------------------------------------

## 3. 即梦 Seedance

### 画面质量（4.8）

**优势**

-   光影层次优秀
-   色彩表现自然
-   构图电影感最佳

### 运动质量（4.6）

**优势**

-   动作自然
-   节奏舒适

**Bad Case**

-   动态表现力 → 动作幅度略保守

### 指令遵循（4.8）

主体、动作、场景、镜头及风格均高度符合 Prompt。

**综合评价**

本轮评测整体表现最佳。

------------------------------------------------------------------------

## 4. CogVideo

### 画面质量（4.2）

**优点**

-   光照自然
-   构图合理

**问题**

-   清晰度偏低
-   局部细节不足
-   边缘略模糊

**Bad Case**

-   图像结构 → 局部细节缺失

### 运动质量（4.1）

**问题**

-   手指运动僵硬
-   笔尖轨迹连续性一般

**归因**

-   运动合理性 → 人体运动异常
-   运动稳定性 → 动作跳变

### 指令遵循（4.5）

主体及动作正确，细节表达不足。

**综合评价**

基础生成能力稳定，但细节与运动质量仍需提升。

# 五、评测结论

本次采用统一评测体系，从**画面质量与美学、运动质量、指令遵循**三个维度对四款主流
T2V 模型进行了横向评测。

主要结论：

-   Seedance 综合能力最优，画质、美学与 Prompt 遵循表现突出。
-   Kling 在运动质量方面表现最佳，人体运动与运镜更自然。
-   Muse Steamer 综合能力均衡，但运动真实性仍有提升空间。
-   CogVideo 基础能力稳定，但图像细节与时序一致性相对较弱。

------------------------------------------------------------------------

# 六、优化建议

## Muse Steamer

-   提升复杂人体运动建模能力。
-   优化精细交互（如笔尖接触纸面）。
-   加强局部纹理与暗部细节恢复。

## Kling

-   控制高光曝光，提升复杂光照场景表现。

## Seedance

-   提升动作幅度与动态表现力。

## CogVideo

-   提高清晰度与局部细节恢复能力。
-   优化时序一致性，减少动作跳变。
